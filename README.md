# Otamaz (Zomato Clone)

A premium, AI-enhanced Zomato clone featuring:
- **Predictive Search**: Smart restaurant search with real-time suggestions.
- **AI Dish Analysis**: Detailed AI-generated descriptions for menu items.
- **Admin Dashboard**: Full management system for restaurants, orders, and users.
- **Responsive Design**: Modern, glassmorphic UI matching the Zomato aesthetic.

## Tech Stack
- **Frontend**: React, Vite, TailwindCSS, Lucide Icons.
- **Backend**: Node.js, Express, MySQL, JWT Auth.
- **AI**: Google Gemini API for dish analysis and similar restaurant recommendations.

## Setup
1. Clone the repository.
2. Run `npm install` in both `client` and `server` folders.
3. Configure your `.env` with DB and Gemini API keys.
4. Run `npm start` (server) and `npm run dev` (client).
